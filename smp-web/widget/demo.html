GoChat Widget Demo

This is a plain HTML page with GoChat embedded via a single script tag. The floating bubble should appear in the bottom-right corner.

Click it to open the chat. Everything is end-to-end encrypted via SimpleX.

Since no data-contact-address is set, the widget runs in mock mode with simulated responses.

Console Commands

Try these in the browser console:

  • GoChat.open() - Open the chat panel
  • GoChat.close() - Close the chat panel
  • GoChat.toggle() - Toggle open/close
  • GoChat.isOpen() - Check if panel is open
  • GoChat.addMessage('Hello!', 'incoming') - Add a message
  • GoChat.reset() - Reset the chat
